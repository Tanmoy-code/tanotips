# **App Name**: SanskritReader

## Core Features:

- Text Translation: Accept Sanskrit text input and translate it into English using AI. LLM is used as a translation tool to intelligently decide when to incorporate different known Sanscrit root meanings.
- Image Translation: Upload an image containing Sanskrit text, and the app will translate it into English. LLM is used as a translation tool to intelligently decide when to incorporate different known Sanscrit root meanings.
- Display Translated Text: Display the translated English text in a clear, readable format.

## Style Guidelines:

- Primary color: Deep saffron (#FF7F50) to evoke the spirit of the Sanskrit language and Indian culture.
- Background color: Light beige (#F5F5DC), a desaturated version of the primary color to ensure comfortable reading.
- Accent color: A muted olive green (#A2BE78), an analogous hue, is used to provide a gentle contrast and highlight interactive elements.
- Headline font: 'Literata', a serif font to add a traditional and elegant look to the headings.
- Body font: 'Inter', a sans-serif font for the body to give it a modern look.
- Use simple, elegant icons that complement the Sanskrit theme. Keep them minimalistic to not distract from the translated content.
- Clean, well-spaced layout for easy readability of both Sanskrit and translated English text.
